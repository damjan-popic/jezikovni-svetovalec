# Maintenance notes

Store lint reports, health checks, and structural decisions here.

Run:

```bash
python tools/wiki_lint.py
```

Useful periodic checks:

- orphan pages,
- missing links,
- missing citations in durable rule pages,
- contradictions that should be moved to `wiki/contradictions.md`,
- recurring student questions that should be filed into `wiki/questions/`.
